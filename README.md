## 📌 Project Overview

This is a comprehensive **Web Application Penetration Testing** report conducted on **bWAAP (Buggy Web Application)**. The assessment was performed as part of the Career Advancement Programme (CAP) in Cybersecurity at Creative IT Institute.

The goal was to identify, exploit, and document vulnerabilities in a controlled environment to demonstrate real-world penetration testing skills.

---

## 🎯 Objectives

- Identify security vulnerabilities in bWAAP
- Exploit vulnerabilities at Low and Medium security levels
- Document findings with Proof of Concept (PoC)
- Provide remediation recommendations

---
